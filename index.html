    Information about ICT, or information, communication, and technology, is the main focus of SLP 1. Internet etiquette, ethics, and safety. ICT Content Development for a Specific Purpose. Utilizing text, graphics, and images to create ICT content for an online environment comes last.

    Lesson 1

    • The Internet's History (a brief overview of the evolution of ICTS in chronological order) So, ICT started in the premechanical period starting around 3000 BCE, then the mechanical period (around 1450 to 1840), and finally the electronic period (1840–1940). In the electronic age, the highlight is the electronic numerical integrator and computer, which were invented by J. Presper Eckert and John Mauchly at the University of Pennsylvania.
    • In this lesson also discussed the overall information about theWorld Wide Web ( WWW ).The program that makes it possible for you to view online files and hypertext documents.
    • URLs (Uniform Resource Locator) are used to formally identify websites, files, and other online resources known as a web address
    • Tim Berners-Lee made the suggestion in 1989
    • There is also Evolution of WWW, Web 1.0 ( which user can only read web ), Web 2.0 ( Read-and-Write Web ) and Web 3.0 ( Semantic Executing Web )
    • Everywhere in this world has dangers and so we us human have to be careful, as well as in the world of cyberspace. Offensive Content, Cyber bullying, Plagiarism, Identity Thefts and Scam are all under Risk and Dangers online.

    • Don't be easy on the internet because on the other side there are dangers. We as the internet users must protect ourselves by taking tips as our guide to be safe and less mistakes/ regrets.
    • Internet-based threats expose people and computer systems to harm online. Malware, Worm, Trojan, Spyware, Spam, and Phishing are all part of Internet Threats.
    • NETIQUETTE = InterNET + EtiQUETTE
    • Etiquette is a code of social behavior
    • Core Rules of Netiquette

    Lesson 3

    • Word Processor is a software application that every student, researcher and computer user uses to create documents.
    • Microsoft Office Word is an example of Word Processor
    • Common MS Word file formats- The manner in which geographic characteristics are stored is determined by file formats. Different formats are used to store raster and vector data. Both raster and vector data are accessible in a variety of forms. Given that software packages seldom support all file types, it is crucial to take the file format of GIS data into consideration.

    • (As students, teachers expect us to be more creative and innovative.)
    • Basic Principles of graphics and layout.
    • Numerous difficulties with organization, networking, optimization, matching, and operation may be solved by studying graphs using a framework. In physical, biological, social, and informational systems, graphs may be used to describe a broad variety of linkages and processes.
